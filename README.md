# Srabon Rivu Portfolio

A responsive static portfolio ready for free hosting on GitHub Pages, Cloudflare Pages or Netlify.

## Personalise before publishing

1. Replace `hello@srabonrivu.com` in `index.html` with your real email.
2. Replace the LinkedIn placeholder URL with your profile URL.
3. Replace placeholder metrics with verified results and reporting periods.
4. Add your photograph and real campaign screenshots.
5. Obtain permission or anonymise client information before publishing it.

## Preview

Open `index.html` directly, or run a local static server in this folder.

## Free deployment

Upload this folder to a GitHub repository and enable GitHub Pages, or import the repository into Cloudflare Pages/Netlify. No build command is required; the publish directory is the project root.
